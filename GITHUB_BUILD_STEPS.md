# Build be lahza APK

1. Upload the extracted project contents to GitHub.
2. Make sure `.github/workflows/android-build.yml` exists.
3. Go to **Actions**.
4. Run **Build be lahza Android APK**.
5. Download the artifact named `be-lahza-v70-connected-production-foundation-debug-apk`.
6. Unzip it and install `app-debug.apk` on your Android phone.
